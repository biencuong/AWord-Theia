# ND30 LLM review checklist

## Categories
- spelling
- punctuation
- wording
- logic
- consistency
- normalization

## Severity
- `critical`: meaning / authority / scope risk
- `warn`: quality or logic weakness needing revision
- `suggest`: optional editorial improvement

## Auto-apply safety
Set `safe_to_apply=true` only if the rewrite does not change substantive meaning.
Set `needs_confirmation=true` when a suggestion touches legal basis, authority, signatory, scope, recipients, deadlines, or assigned responsibility.
