# ND30 validation checklist

Run this checklist before final delivery.

## 1. Document-family correctness

- Is the selected document type the correct one for the purpose?
- If the user only asked generally, did you explain the chosen form?

## 2. Mandatory metadata

- Parent agency present if needed
- Issuing agency present
- Number/symbol present and plausible
- Location/date line present and formatted correctly
- Title/summary block appropriate for the family

## 3. Body logic

- Legal bases included where the family normally requires them
- Article structure used for decisions/resolutions where appropriate
- `Kính gửi` only where appropriate
- `Nơi nhận` formatted and complete
- Signer authority prefix used only when justified

## 4. Formatting

- A4, portrait unless justified otherwise
- Margins inside ND30 ranges
- Times New Roman, Unicode
- Body size around 13–14 pt
- National title block and agency block visually balanced
- Signature block on the right, nơi nhận on the left

## 5. Writing quality

- No chatty language
- Proper capitalization
- Correct abbreviations
- Dates, names, and numbers preserved exactly

## 6. Layout risk

- No clipped text
- No over-wide tables
- No strange page breaks
- Signature area has enough space
- Rendered pages look clean if rendering is available
