# Internal template library

This skill includes internal templates derived from real agency documents. They are grouped as:

- `assets/templates/internal/` → active DOCX templates and their metadata
- `assets/profiles/internal/` → extracted DOCX format profiles
- `assets/internal-corpus/` → supporting source documents, PDFs, and legacy files

Use active internal templates when the user wants institutional closeness, but still audit the output against ND30.
If a template carries warnings, tell the user before using it in replicate mode.
