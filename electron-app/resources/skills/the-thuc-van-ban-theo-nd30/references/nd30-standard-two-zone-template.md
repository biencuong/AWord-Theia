# Standard 2-zone template

The standard baseline file is `assets/templates/standards/mau-tieu-ngu-va-noi-nhan-chuan.docx`.

Use it as the reference for these sensitive blocks:
- issuing agency + rule line
- quốc hiệu – tiêu ngữ + rule line
- Kính gửi or named-title placement
- Nơi nhận block
- signature block

Important: the rules below the issuing agency and below “Độc lập - Tự do - Hạnh phúc” are **layout lines**, not hand-typed underscores.
